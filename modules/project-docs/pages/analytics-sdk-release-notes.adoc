[#latest-release]
== Java Analytics SDK 1.0 Releases

We always recommend using the latest version of the SDK -- it contains all of the latest security patches and support for new and upcoming features.
All patch releases for each dot minor release should be API compatible, and safe to upgrade;
any changes to expected behavior are noted in the release notes that follow.


=== Version 1.0.0 (10 July 2025)

https://docs.couchbase.com/sdk-api/analytics-dotnet-client-1.0.0[API Reference]

This is the first General Availability (GA) release of the new Analytics Java SDK. 
It supports executing queries against Enterprise Analytics clusters, with additional features planned for future releases.
