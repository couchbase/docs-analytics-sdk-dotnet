== Getting the SDK


Couchbase publishes all stable artifacts to https://www.nuget.org/profiles/couchbase[NuGet].

Install using your preferred method:

=== .NET CLI

[source,shell,subs="attributes+"]
----
dotnet add package Couchbase.AnalyticsClient --version {sdk_current_version}
----

=== Add to .csproj

[source,xml,subs="attributes+"]
----
<ItemGroup>
  <PackageReference Include="Couchbase.AnalyticsClient" Version="{sdk_current_version}" />
</ItemGroup>
----
